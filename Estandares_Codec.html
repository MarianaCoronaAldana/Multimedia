Codecs de video

Un códec de video es el lenguaje informático con el que está escrita la información del video. Así como en el mundo del audio existe el .wav, .mp3 o el .FLAC. En video también hay diferentes códecs! El término “coder” se refiere a codificar y “decoder” a descodificar. Es un código que realiza su función cuando el sistema operativo o software lo necesita. Este código transforma las señales digitales de audio y video en un formato que puede reproducirse.
Dicho de otro modo, un Codec codifica y comprime los datos de un archivo, vídeo o audio para agilizar su transferencia o para que ocupe menos espacio. Al reproducir o editar el archivo, se descomprime. Esto quiere decir que estamos ante una palabra compuesta ya que el Codec desempeña dos funciones: comprimir y descomprimir.
Lo primero lo hace para que el archivo sea más portable y, lo segundo, para proporcionar todos los datos del archivo. Este código es clave en los vídeos. Lo que pasa es que existen Codecs que comprimen demasiado el archivo, mermando la calidad de reproducción.
Hay muchos Codecs diferentes, cada uno con sus ventajas y desventajas. Se trata de encontrar el que mejor se adapta a las necesidades del usuario. Vamos a ver algunos:

  • MPEG-4: Codec muy popular es el MPEG-4 para vídeo. Usa una compresión mucho mejor que MPEG-1 y su calidad es muy buena. En este formato, hay muchos Codecs. Por ejemplo, el H.264, el que se usa para codificar vídeo para Blu-Ray. Destaca por la gran flexibilidad que ofrece, tanto para resoluciones altas como bajas.

  • H265: El nuevo estándar H.266, también conocido como VVC (Versatile Video Coding), sirve para comprimir y descomprimir los archivos de vídeo para que ocupen menos espacio cuando se almacenan en un disco duro y menos ancho de banda cuando se transfieren a través de una red.

  • Daala: Daala es el nombre de trabajo actual de un códec de vídeo de próxima generación que está siendo desarrollado por la Fundación Xiph.Org. Daala pretende ser un códec de ultra alta eficiencia de próxima generación para casos de uso similares a High Efficiency Video Coding (HEVC o H.265)

  • Theora: Theora es un códec de vídeo libre que está siendo desarrollado por la Fundación Xiph.Org, como parte de su proyecto Ogg. Basado en el códec VP3 donado por On2 Technologies, Xiph.Org lo ha refinado y extendido dándole el mismo alcance futuro para mejoras en el codificador como el que posee el códec de audio Vorbis.

  • WMV: (Windows Media Video) Este es un codec desarrollado por Microsoft que intenta trae una genial visualización WMV. ​Cabe aclarar que típicamente puede referirse a un códec de video y a un archivo contenedor, sin embargo, aquí estaremos hablando del codec. Este codec le permite abrir otros contenedores de archivos como AVI. Ya casi no se usa por la aparición de otros formatos más modernos como FLV.
